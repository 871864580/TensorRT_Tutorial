NVIDIA TensorRT 2.0 Download

NVIDIA TensorRT is a high-performance deep learning inference for production deployment.

Thank you for participating in the early access program. The goal of this phase of the program is to gather feedback from you on the usability, performance and functionality of the new INT8 inference capabilities of TensorRT.

This new functionality allows you to start with a neural network that you trained in single-precision floating point (called FP32) and use TensorRT to create an inference solution that takes advantage of the high-performance eight bit integer (INT8) capability of the Pascal P4 and P40 cards (and similar). This technique uses INT8 values for both the weights and activations of the layers in the neural networks. INT8 numbers have a more limited dynamic range so you will need to run a calibration process using TensorRT that selects internal scaling factors so that each layer sees the appropriate signal from the layer before. Please see the user guide for a detailed discussion of this calibration process.

This phase of the early access program is focused very specifically around INT8 inference -- there are additional features (the new customer layer API, FP16 improvements) that we will be adding into TensorRT 2 which will be unveiled in an EA program refresh in approximately 1-2 months. If you are interested in those capabilities we advise you to wait until the second phase of EA testing is announced.

The following Pascal based devices have the INT8 instruction that this version of TensorRT will help you take advantage of:
Tesla P4, Tesla P40, GeForce TitanX Pascal, GeForce GTX 1080, DRIVE PX 2 dGPU

If you have access to one or more of those and are interested in INT8 then we encourage you to test TensorRT 2 during this EA. If you do not have access to one of these devices we recommend you instead test TensorRT (GIE) 1.

A few notes to be aware of before downloading:

  1. We renamed TensorRT between version 1 and version 2. It was previously called GIE. The package names have changed. TensorRT is a meta package that installs the nvinfer and nvinfer-devel packages which provide the library and developmental resources (header files, documentation, etc..)
  2. If you have GIE already installed TensorRT 2 will install next to your existing GIE installation. GIE should continue to be the default. Please link specifically against libnvinfer.so.2 to test TensorRT 2.
Quick Start Instructions

  1. Verify that you have the CUDA toolkit installed, release 8.0 .
  2. Download the TensorRT debian package (below)
  3. Install TensorRT from the debian package:

      sudo dpkg -i nv-tensorrt-repo-ubuntu1404-7-ea-cuda8.0_2.0.1-1_amd64.deb
      sudo apt-get update
      sudo apt-get install tensorrt-2

  4. Verify your installation:

      dpkg -l | grep tensorrt-2

    and you should see:

      tensorrt-2 2.0.0-1+cuda amd64 Meta package of TensorRT

      dpkg -l | grep nvinfer2

    and you should see:

      libnvinfer2 2.0.0-1+cuda amd64 TensorRT runtime libraries

Documents are included in the install package or can downloaded separately: TensorRT 2.0 EA Documentation

  • TensorRT user guide can be found at: /usr/share/doc/gie/doc/User Guide.html
  • TensorRT API manual can be found at: /usr/share/doc/gie/doc/API/index.html
  • Sample code can be found at: /usr/src/gie_samples/samples
